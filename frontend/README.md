# 🧠 SmartCV

SmartCV is a full-stack web application that uses AI to analyze resumes and provide instant feedback. It generates a score, highlights strengths and weaknesses, and delivers actionable feedback to help users refine their profiles and become more job-ready.

---

## 🚀 Features

- 📄 Paste and analyze your CV instantly  
- 🧠 AI-powered resume scoring (0–100)  
- 💡 Strengths and weaknesses breakdown  
- ✨ Improved CV suggestions  
- 📊 ATS-friendly insights  
- 🌙 Dark / Light mode UI  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Axios  
- CSS3  

### Backend
- Node.js  
- Express.js  
- Google Gemini API  
- CORS  

---

## 🧠 How It Works

1. User pastes their CV into the input box  
2. Frontend sends the data to the backend API  
3. AI analyzes the resume and returns insights  
4. UI displays:
   - Resume score  
   - Strengths  
   - Weaknesses  
   - Improved CV
